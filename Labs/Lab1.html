Laboratorio 1: Introducción a las aplicaciones web, HTML5 y ciclo de vida de los sistemas de información

Preguntas

  • ¿Cuál es la diferencia entre Internet y la World Wide Web?
    El internet es una inmensa conexion de computadoras entre si alrededor de todo el mundo, y WWW es una coleccion de paginas que trabajan sobre el internet.
  • ¿Cuáles son las partes de un URL?
    Protocolo, Dominio, Path y Parametros
  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?
    GET: Solicita informacion a un recurso en especial donde los datos no se pueden modificar.
    HEAD: Se utiliza para comprobar si el recurso se encuentra antes de hacer un GET.
    POST: Envia datos a un servidor.
    PUT: Edita un recurso ya existente en un servidor.
    PATCH: Aplica modificaciones parciales a un recurso.
    DELETE: Borra un recurso en especifico.
  • ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?
    POSt, ya que este es mas seguro
  • ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?
    GET
  • Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?
    Que todo salio bien
  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?
    No, es del usuario, y del desarrollador solo si es algo que no se encontro dentro de la pagina.
  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?
    Si, ya que es un fallo con el servidor.
  • ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.
    Que es una etiqueta vieja que se cambio por otra, pero aun es soportada en los navegadores.
    La etiqueta "center" es un ejemplo.
  • ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?
    HTML5 tiene un control de errores para que su sintexis sea compatible con todas las implementaciones.
  • ¿Qué componentes de estructura y estilo tiene una tabla?
    Table, THEAD, TFOOT, TBODY, TR,
    width %Length; #IMPLIED -- anchura de la tabla --
    border %Pixels; #IMPLIED -- controlar el borde que rodea a la tabla --
    frame %TFrame; #IMPLIED -- qué partes del borde representar --
    rules %TRules; #IMPLIED -- líneas de división entre filas y columnas --
    cellspacing %Length; #IMPLIED -- espaciado entre celdas --
    cellpadding %Length; #IMPLIED -- espacio dentro de las celdas --
  • ¿Cuáles son los principales controles de una forma HTML5?
    form, fieldset, legend, textarea, label, button e input.

  • ¿Cuál es el ciclo de vida de los sistemas de información?
    Nacimiento, Desarrollo, Operaciones, Mantenimiento y Muerte.
  • ¿Cuál es el ciclo de desarrollo de sistemas de información?
    Planificacion, Definicion de requisitos, Diseño, Desarrollo y Pruebas.
